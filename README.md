# Habit Tracker

Needs backend .env file



## Installation

```bash
  npm install 
  cd front 
  npm install
  cd ..
  cd backend
  npm install
```
## Running 

```bash
    cd front 
    npm start
```
```bash
    cd backend 
    node server.js
```
